Bug Bounty PoC: CORS Misconfiguration

Target: uat-api.paylution.com (Hyperwallet UAT API)
CWE: CWE-942 — Overly Permissive Cross-Origin Resource Sharing
Scope: Authorized bug bounty research

Vulnerability

uat-api.paylution.com reflects any Origin header with Access-Control-Allow-Credentials: true. The login endpoint POST /rest/internal/v1/jwts is reachable cross-origin — an attacker page can submit credentials and read the JWT from the response, gaining full API access to the victim's merchant account.

Live Demo

This page is hosted on sufyan-aslam.github.io — a completely different origin from uat-api.paylution.com. Enter Hyperwallet UAT credentials below. The page will POST them cross-origin and read the JWT from the response.
